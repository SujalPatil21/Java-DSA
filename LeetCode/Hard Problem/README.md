# Hard Problems – LeetCode

## Description
This directory contains hard-level problems categorized by topic.  
These problems require deep problem-solving ability, strong pattern recognition, and optimized approaches under constraints.

The focus is on mastering complex logic and writing highly efficient solutions suitable for top-tier technical interviews.

---

## Categories
- Arrays

## Approach & Strategy
- Analyze constraints before jumping into coding  
- Identify hidden patterns and reduce brute force  
- Break problems into subproblems  
- Optimize aggressively (time and space)  
- Validate edge cases thoroughly  

---

## Concepts Covered
- Advanced Data Structures  
- Dynamic Programming  
- Greedy Algorithms  
- Binary Search on Answer  
- Sliding Window (Advanced)  
- Monotonic Stack / Queue  
- Graph-based Thinking (when applicable)  

---

## Goal
To reach a level where hard problems become structured rather than overwhelming, and solutions are derived logically under interview pressure.
