# Hard Recursion Problems – LeetCode (Java)

This folder contains **hard-level LeetCode problems based on recursion and backtracking**, implemented in Java.

These problems focus on advanced recursive thinking, deep backtracking patterns, optimization techniques, and complex decision-tree exploration commonly asked in high-level coding interviews.

---

## Concepts Covered
- Advanced recursion
- Backtracking
- Recursive tree traversal
- Constraint-based search
- Combination and permutation generation
- Pruning techniques
- Divide and conquer
- State-space exploration
- Recursive optimization

---

## Purpose
- Strengthen advanced recursion skills  
- Master hard-level backtracking problems  
- Improve recursive problem-solving speed and accuracy  
- Build foundations for dynamic programming and graph recursion  
- Prepare for product-based company interviews
