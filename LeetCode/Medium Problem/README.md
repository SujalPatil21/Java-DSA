# Medium Problems – LeetCode

## Description
This directory contains medium-level problems categorized by topic.  
These problems form the core of technical interview preparation and require a balance of logic, pattern recognition, and optimization.

The focus is on transitioning from basic solutions to efficient approaches while improving problem-solving speed and accuracy.

---

## Categories
- Array  
- Recursion  
- String  

---

## Approach & Strategy
- Start with a clear understanding of the problem  
- Build an initial working solution (brute force if needed)  
- Identify patterns and optimize step-by-step  
- Focus on reducing time and space complexity  
- Practice writing clean, interview-ready code  

---

## Concepts Covered
- Sliding Window  
- Two Pointer Technique  
- Hashing (HashMap / HashSet)  
- Recursion & Backtracking  
- Binary Search  
- Greedy Algorithms  
- Basic Dynamic Programming  

---

## Goal
To consistently solve medium-level problems efficiently and build the capability to handle real interview questions with confidence.
