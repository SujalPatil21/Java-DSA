# Maths Problems – Medium Level

## Description
This directory contains medium-level mathematical problems solved using Java.  
These problems focus on applying mathematical logic, number properties, and optimized computation techniques.

The goal is to move beyond brute force and develop efficient, pattern-based solutions.

---

## Approach & Focus Areas
- Understanding mathematical patterns and properties  
- Reducing brute force using optimized logic  
- Handling large numbers and edge cases  
- Applying formulas instead of iterative solutions  
- Writing efficient and clean implementations  

---

## Concepts Used
- Binary Exponentiation  
- Number Theory Basics  
- Modular Arithmetic  
- Prime Numbers & Factors  
- Mathematical Observations  
- Bit Manipulation (when applicable)  

---

## Goal
To develop strong mathematical intuition for problem-solving and efficiently handle computation-heavy interview questions.
