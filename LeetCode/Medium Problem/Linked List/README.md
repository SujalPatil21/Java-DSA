# Medium Linked List Problems – LeetCode (Java)

This folder contains **medium-level LeetCode problems based on Linked Lists**, implemented in Java.

These problems focus on advanced pointer manipulation, efficient traversal techniques, linked list modifications, and common interview patterns frequently asked in technical interviews.

---

## Concepts Covered
- Singly Linked List
- Doubly Linked List
- Fast and Slow Pointers
- Dummy Nodes
- In-place Reversal
- Linked List Traversal
- Node Deletion
- Node Insertion
- Merging Linked Lists
- Cycle Detection
- Pointer Manipulation

---

## Purpose
- Strengthen Linked List problem-solving skills
- Master common interview patterns
- Improve pointer handling and debugging
- Learn efficient linked list modifications
- Prepare for coding interviews and placements
