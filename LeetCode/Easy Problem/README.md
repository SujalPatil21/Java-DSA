# Easy Problems – LeetCode

## Description
This directory contains easy-level problems categorized by topic.  
The focus is on building strong fundamentals, improving coding consistency, and developing a clear understanding of basic problem-solving patterns.

Each category targets a specific concept to ensure structured learning instead of random practice.

---

## Categories
- String Problems  
- HashMap Problems  
- Array Problems  

---

## Approach & Strategy
- Start with understanding the problem clearly  
- Write a simple brute-force solution first  
- Improve step-by-step toward an optimal solution  
- Focus on clean and readable code  
- Practice consistency over intensity  

---

## Concepts Covered
- Basic Data Structures  
- String Operations  
- Arrays and Traversal  
- Hashing Fundamentals  
- Two Pointer Technique (Intro)  

---

## Goal
To build a solid foundation in Data Structures and Algorithms, enabling a smooth transition to medium and hard-level problems.
