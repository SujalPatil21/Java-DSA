# Easy Linked List Problems – LeetCode (Java)

This folder contains **easy-level LeetCode problems based on Linked Lists**, implemented in Java.

These problems focus on building strong fundamentals of linked list traversal, node manipulation, insertion, deletion, and pointer handling.

---

## Concepts Covered
- Singly Linked List
- Doubly Linked List
- Node traversal
- Insertion operations
- Deletion operations
- Middle node finding
- Pointer manipulation
- List reversal basics
- Length calculation

---

## Purpose
- Strengthen Linked List fundamentals  
- Improve pointer handling skills  
- Understand node-based data structures  
- Build a foundation for advanced Linked List problems  
- Prepare for coding interviews and placements  

---

## Notes
- Each problem has its own folder
- Every problem includes:
  - Java solution
  - Individual `README.md`
  - Test case screenshot
  - Accepted submission screenshot
- Focus is on:
  - Clean logic
  - Proper traversal
  - Efficient pointer usage

---

## Common Problem Types
- Middle of the Linked List
- Reverse Linked List
- Remove Duplicates
- Merge Two Sorted Lists
- Delete Node in Linked List
- Palindrome Linked List
- Cycle Detection
- Linked List Traversal
